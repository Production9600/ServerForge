# Image Creation To-Do List for Server Forge Clan Page

This file lists the images needed to complete the website's design. Each theme requires its own set of images to create a unique look and feel.

---

### 1. Theme: 7 Days to Die (`themes/7d2d/`)

- **Logo (`logo.png`):**
  - **Style:** Apocalyptic, grungy, distressed. Text should be your clan name. Consider a biohazard or skull symbol.
  - **Dimensions:** Approx. 300x80 pixels.
  - **Save Location:** `server-forge/web_content/themes/7d2d/images/`

- **Header Background (`header_bg.jpg`):**
  - **Style:** Dark, gritty texture like cracked concrete or rusty metal.
  - **Dimensions:** 1200x150 pixels.
  - **Save Location:** `server-forge/web_content/themes/7d2d/images/`
